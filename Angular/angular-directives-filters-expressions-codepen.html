Expressions

Angular expressions are Javascript-like snippets that are usually placed in bindings.

  • Expressions can be used as an argument for a directive
  • Expressions can be used to display information on the page like: cohort number {{ cohort }} is the best!
  • You can add numbers: 1 + 2 = {{1 + 2}}
  • You can concatenate strings: My name is {{"Alexander" + " Hamilton"}}
  • You can get evaluate the length of an array - [99,43,22] {{ [99,43,22].length }} is items long.
  • You can't access variables outside of the Angular scope, so you can't access the school name here - {{school}}

Filters

Angular filters are bits of code that format data before displaying it.

  • You can limit the number of characters shown: {{ 3.14159265359 | limitTo: 3 }}
  • You can display currency - {{3.14159265359 | currency }}
  • You can {{ "make text bigger" | uppercase }}
  • You can chain filters together like here where an array is filtered to include only values with the number 2 in them and then results are limited to give us only 3 results. {{ [1,12,13,14,25,200,1002] | filter: 2 | limitTo: 3}}
